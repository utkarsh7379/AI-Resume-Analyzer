# AI Resume Analyzer

An AI-powered Resume Analyzer developed using MERN Stack and Python NLP.

## Features

- Resume Upload (PDF)
- ATS Score Analysis
- Skill Extraction
- Missing Skill Detection
- Resume Improvement Suggestions
- Dashboard Analytics

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI/NLP

- NLP
